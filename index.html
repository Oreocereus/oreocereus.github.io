Garden Planner App.

Purpose

This is a python coded desktop app with a GUI, intended to be used to help the user find the right tree for them. First it prompts them to select a size of tree, then a season, then a decorative feature, it then shows a list of suitable trees that match the criteria. Once the user selects a tree from the list, it will show a picture of the tree and a brief description of it.

Methods used

  • Tkinter for the GUI
  • Functions to control the buttons
  • Nested dictionaries to contain the tree information























Card Game.

Purpose

This is a card game using a 52 card deck plus jokers, coded in python with a GUI. It is intended to be a casino style game where cards are drawn from a shuffled deck and there are several special cards which when drawn, reshuffle in all the red cards that have been previously drawn. The user can bet on whether the next card will be a a black card or not. As the red cards are reshuffled into the deck when a special card is drawn, it becomes harder to bet correctly if the next card is black, therefore the prizes for betting correctly later in the game increase.






















Methods used

  • Tkinter for the GUI
  • Saving feature which saves the state of the deck as a json
  • Updating lists for card/deck management
  • Betting mechanism
  • Various functions to control different mechanics e.g. draw_card function



ISS Tracker.

Purpose

A program which uses the International Space Station API and the user's location to check if the ISS is overhead. If it is, an email is sent to the user to tell them to look up at the ISS. It also checks the users time and uses sunrise and sunset data to check if it is daytime or nighttime. If it is day it sends an email saying the ISS is overhead but you will not be able to see it. If it is nighttime it just sends an email saying the ISS is overhead.

Methods used

  • API to track ISS and get sunrise/sunset data
  • smtplib module to send automated email notifications
  • Environment secured credentials to login and uses sendmail to conditionally send email
  • Uses a function to determine if it is daylight or nighttime using sunrise/sunset data
